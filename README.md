# Python ACES adaptation - Frontend React SPA

## Version 0.0.1-alpha

Currently in alpha testing, not for production use.

## Installation

```sh
git clone https://github.com/arkmoon/PythAces-frontend-react
cd PythAces-frontend-react
yarn install
```
## Configuration & Usage

Local development:
```sh
yarn start
```

Production build:
```sh
yarn build
```
